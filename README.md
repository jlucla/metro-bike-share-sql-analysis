# LA Metro Bike Share Usage Analysis

## Overview

This project analyzes bike share usage patterns using trip data from LA Metro Bike Share Q4 (October–December 2025).

The goal was to explore how total trips taken vary across stations, time, and category of trip. SQL queries were used to conduct exploratory analysis, followed by the development of an interactive Excel dashboard.

## Tools 

**Excel**: PivotTables, slicers, VLOOKUP functions, dashboard design

**SQL (SQLite)**: aggregations, filtering, grouping, window functions

## Questions Explored

- When are bike share trips most and least popular?

- How does usage vary by time of day and month?

- Which stations have the highest and lowest demand?

- How do one-way vs round trips differ across time?

- Which stations are associated with longer trips?

## Key Insights

- Bike share usage peaks during evening commute hours

- One-way trips dominate over round trips (~80%)

- Stations with the highest usage differ from those with the longest trip durations
